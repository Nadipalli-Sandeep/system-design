# System Design – End-to-End Learning by Building

This repository documents my journey of learning **system design from scratch by building real systems end-to-end**.

## What this repository contains

### Phase 1 – Authentication System
- User Signup
- Duplicate User Prevention
- Login System
- Password Hashing (bcrypt)
- Stateless Authentication (JWT)
- Protected Routes
- Middleware-based Authorization

## System Design Concepts Covered
- Client–Server Architecture
- REST API Design
- Read vs Write Paths
- Stateless vs Stateful Systems
- Authentication vs Authorization
- Failure Scenarios
- Security by Design

## Tech Stack
- Node.js
- Express.js
- bcrypt
- JSON Web Tokens (JWT)
